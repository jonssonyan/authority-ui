# 权限管理系统前端

基于SpringBoot和Vue.js的权限管理系统前端

本项目为前后端分离架构，本项目的[后端仓库地址](https://github.com/jonssonyan/authority)

## 快速开发

1. 克隆项目

    ```shell
    git clone https://github.com/jonssonyan/authority-ui
    ```

2. 安装依赖

    ```bash
    cd authority-ui
    yarn
    ```

3. 本地开发

    ```bash
    yarn run serve
    ```

打开 http://localhost:8888 预览并调试

## 技术栈

- JavaScript
- Vue.js
- Element
- Axios
- Node v14.17.0
- yarn 1.22.21
