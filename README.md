# 权限管理系统

一个前后端分离的用户权限管理系统

# 开发工具

1. WebStorm 2019.3.5

# 框架/技术

1. Vue.js
2. Element
3. axios

# 默认账号

- 管理员：admin 123456

- 普通用户：user1 123456

这些数据是存在数据库里的，注册一个用户，然后在数据库里把角色改为1（管理员角色）即可
